## 📚 P0优先级文档源详解

### 一、立法解释（LEGISLATIVE_INTERPRETATION）⭐⭐⭐⭐⭐

#### 1. 基本信息
```yaml
实体类型: LEGISLATIVE_INTERPRETATION
发布机关: 全国人大常委会
法律效力: 与法律具有同等效力（最高效力）
数据量: 少（约50-100个）
唯一标识: LEGISLATIVE_INTERPRETATION_NUMBER
```

#### 2. 特征识别
**标题特征**：
- 《全国人民代表大会常务委员会关于〈XXX法〉第X条的解释》
- 必须由全国人大常委会作出
- 通常针对具体法条进行解释

**示例**：
```
《全国人民代表大会常务委员会关于〈中华人民共和国刑法〉第九十三条第二款的解释》
《全国人大常委会关于〈中华人民共和国香港特别行政区基本法〉第一百零四条的解释》
```

#### 3. 新增关系类型

##### ⭐ legislative_interprets（立法解释关系）- 最高优先级
```yaml

含义: 立法解释解释法律条文，效力等同法律
强度: 10分（与法律同等效力）

示例:
{
  "source": "全国人大常委会关于刑法第九十三条第二款的解释",
  "target": "《刑法》第九十三条第二款",
  "description": "立法解释明确了'其他依照法律从事公务的人员'的具体范围",
  "strength": 10
}
```

**识别关键词**：
- "全国人大常委会关于...的解释"
- "立法解释明确"
- "解释如下"

##### authorizes（授权关系）
```yaml
定义: [LAW] -> [LEGISLATIVE_INTERPRETATION]
含义: 法律授权进行立法解释
强度: 9分
```

### 二、法律批复/答复（JUDICIAL_REPLY）⭐⭐⭐⭐

#### 1. 基本信息
```yaml
实体类型: JUDICIAL_REPLY
发布机关: 最高人民法院、最高人民检察院
法律效力: 具有指导意义，各级法院/检察院应遵照执行
数据量: 中等（数百个）
```

#### 2. 特征识别
**标题特征**：
- 《最高人民法院关于XXX问题的批复》
- 《最高人民检察院关于XXX问题的答复》
- 通常针对下级法院/检察院的请示

**示例**：
```
《最高人民法院关于审理涉及农村集体土地行政案件若干问题的批复》
《最高人民检察院关于强奸罪既遂标准有关问题的答复》
```

#### 3. 新增关系类型

##### replies_to（批复关系）

```yaml
定义: [JUDICIAL_REPLY] -> [LEGAL_QUESTION]
```

##### guides_application（指导适用关系）

```yaml
  "strength": 9
```

##### binding_on（约束关系）

```yaml
定义: [JUDICIAL_REPLY] -> [COURT/PROCURATORATE]
```

### 四、国际条约（INTERNATIONAL_TREATY）⭐⭐⭐⭐

#### 1. 基本信息
```yaml
实体类型: INTERNATIONAL_TREATY
签署方: 中国与其他国家/国际组织
法律效力: 经批准的条约优先于国内法（除宪法）
数据量: 中等（数百个）
唯一标识: TREATY_NUMBER
```

#### 2. 特征识别
**类型**：
- 双边条约：《中华人民共和国和XX国关于XX的协定》
- 多边条约：《联合国反腐败公约》、《巴黎气候协定》

**重要条约示例**：
```
《联合国国际货物销售合同公约》（CISG）
《保护工业产权巴黎公约》
《伯尔尼保护文学和艺术作品公约》
《承认及执行外国仲裁裁决公约》（纽约公约）
```

#### 3. 新增关系类型

##### implements（实施关系）
```yaml
定义: [LAW/REGULATION] -> [INTERNATIONAL_TREATY]
含义: 国内法实施国际条约
强度: 9分

示例:
{
  "source": "《涉外民事关系法律适用法》",
  "target": "《联合国国际货物销售合同公约》",
  "description": "国内法规定了公约在中国的适用规则",
  "strength": 9
}
```

##### accedes_to（加入/批准关系）
```yaml
定义: [CHINA] -> [INTERNATIONAL_TREATY]
含义: 中国加入、批准或签署条约
强度: 10分

示例:
{
  "source": "中华人民共和国",
  "target": "《联合国反腐败公约》",
  "description": "中国于2005年10月27日批准该公约",
  "strength": 10
}
```

##### prevails_over（优先适用关系）
```yaml
定义: [INTERNATIONAL_TREATY] -> [LAW]
含义: 条约优先于国内法（除宪法）
强度: 10分

示例:
{
  "source": "《联合国国际货物销售合同公约》",
  "target": "《合同法》相关规定",
  "description": "涉及国际货物买卖时，公约优先于合同法适用",
  "strength": 10
}
```

#### 4. 应用场景

**场景：涉外合同纠纷的法律适用**
```
用户查询：中美之间的国际货物买卖合同适用什么法律？

知识图谱查询路径：
```

```
├─ arbitrated_by → 仲裁委员会          ├─ adjudicated_by → 法院
├─ enforced_by → 法院强制执行          ├─ 自然具有执行力
├─ vacated_by → 可能被法院撤销         ├─ appeals_from → 上诉
└─ 一裁终局                            └─ 二审终审

相同点：
- similar_to（都是争议解决方式）
- 都具有法律约束力
- 都可强制执行
```

### 六、行政处罚决定书（ADMINISTRATIVE_PENALTY）⭐⭐⭐⭐

#### 1. 基本信息
```yaml
实体类型: ADMINISTRATIVE_PENALTY
发布机关: 各级行政机关（市场监管、环保、税务、公安等）
法律效力: 对被处罚人有约束力
数据量: 极大（百万级）
唯一标识: ADMINISTRATIVE_PENALTY_NUMBER
```

#### 2. 主要类型
```

税务处罚：偷税、漏税、欠税
公安处罚：治安管理处罚
海关处罚：走私、违规进出口
```

## 📝 提示词优化要点

### 1. 文档类型识别增强

```markdown
### 新增文档类型识别规则：

#### 立法解释识别：
- 关键词："全国人大常委会关于...的解释"
- 发布机关：必须是全国人大常委会
- 效力标注：与法律同等效力

#### 法律批复识别：
- 关键词："批复"、"答复"
- 发布机关：最高法、最高检
- 针对性：通常答复下级请示

#### 会议纪要识别：
- 关键词："会议纪要"、"座谈会纪要"
- 发布机关：最高法
- 特征：统一审判尺度

#### 国际条约识别：
- 关键词："公约"、"协定"、"条约"
- 多语言：通常有中英文对照
- 批准信息：注明批准/加入时间

#### 仲裁裁决识别：
- 案号特征："贸仲"、"仲裁"
- 发布机关：仲裁委员会
- 终局性：一裁终局

#### 行政处罚识别：
- 文号特征：包含"处罚"字样
- 发布机关：行政机关
- 内容：处罚事实、依据、决定

#### 地方法规识别：
- 地域特征：省市名称
- 制定机关：地方人大
- 适用范围：本行政区域
```
